# FocusBoard 预置缺陷

本文件只用于课程演示说明，不应作为 SpecProbe 的输入需求。

| 编号 | 缺陷 | 预期证据 |
| --- | --- | --- |
| DEMO-001 | `/api/tasks` 固定返回 HTTP 500 | 页面探测状态码、浏览器网络请求、控制台错误 |
| DEMO-002 | 空白或仅空格的任务标题仍会被添加 | 浏览器交互结果、任务列表截图 |
| DEMO-003 | 勾选任务后“已完成”统计始终为 0 | 操作前后 DOM 文本和截图 |
| DEMO-004 | “已完成”筛选按钮仍显示全部任务 | 点击步骤、任务列表 DOM 和截图 |
| DEMO-005 | 新增任务没有写入 localStorage | 刷新前后任务列表差异 |

当前 SpecProbe 课程版本可以稳定发现 `DEMO-001`，也能为需求质量问题生成 Issue 和修复提案。其余交互缺陷用于说明 Playwright 深度执行器的后续扩展价值。
